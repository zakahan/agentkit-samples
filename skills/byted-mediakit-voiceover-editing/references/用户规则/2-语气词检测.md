# 语气词检测

## 语气词列表

```javascript
const fillerWords = ['嗯', '啊', '哎', '诶', '呃', '额', '唉', '哦', '噢', '呀', '欸'];
```

## 结尾确认词/反问词

句尾出现时可标记待删，按用户偏好处理：

```javascript
const endingFillerWords = ['对吧', '是吧', '嘛', '呢', '哈'];
```

| 示例 | 处理 |
|------|------|
| 总会有各种各样的口癖，对吧？ | 可删除「对吧」 |
| 就是这样，是吧？ | 可删除「是吧」 |

## 删除边界

```
错误：删语气词时间戳 (语气词.start - 语气词.end)
      → 可能删掉前面字的尾音

正确：从前一个字的 end 到后一个字的 start
      → (前字.end - 后字.start)
```

## 用户偏好

保留适量"嗯"作为过渡，不要全删。

## 指代完整性（举例结构保护）

当 嗯啊呃 等作为**举例内容**出现（如「比如说，嗯啊呃这种」）时，其为举例主体，**不得删除**，否则「这种」无指代。

| 原文 | 错误做法 | 正确做法 |
|------|----------|----------|
| 比如说，嗯啊呃这种，还有说错了重新说的 | 删「嗯啊呃」→「比如说，这种，」（语义断裂） | **保留整段**（嗯啊呃为举例内容） |
| 像这种、这类、这些（前有语气词作举例） | 单独删语气词 | 保留或整段标记 |

**处理规则**：若 嗯啊呃 等与「这种/这类/这些」绑定，作为举例内容（如口癖、语气词举例），**必须保留**，不得按语气词删除。
