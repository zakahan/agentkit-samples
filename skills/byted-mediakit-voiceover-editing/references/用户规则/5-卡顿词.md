# 卡顿词

## 模式

同一个词连续说 2-3 次：

```javascript
const stutterPatterns = [
  '那个那个',
  '就是就是',
  '然后然后',
  '这个这个',
  '所以所以',
  '去去',
  '去去去',
];
```

## 删除策略

删前面，保留最后一个。

```
原文: "那个那个我想说"
删除: "那个"
保留: "那个我想说"
```

## 不视为卡顿（例外）

量词重叠（一个、一件、一句 等）多为强调，**不纳入卡顿词删除**：

| 原文 | 处理 |
|------|------|
| 一个一个地 | 保留 |
| 我最近做的一个一个口播 | 保留（量词强调） |
| 一件一件检查 | 保留 |
