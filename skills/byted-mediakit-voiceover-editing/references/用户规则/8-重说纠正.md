# 重说纠正

## 模式

说错后立即纠正，删除前面错误的部分。

### 1. 部分重复

前后词语有重叠但不完全相同：

| 原文                 | 删除         |
| -------------------- | ------------ |
| 你再关你关掉         | "你再关"     |
| 怎么让它去有个更大的 | "怎么让它去" |

### 2. 否定纠正

用否定词纠正刚说的：

| 原文       | 删除   |
| ---------- | ------ |
| 它是它不是 | "它是" |
| 可以不可以 | "可以" |

### 3. 词被打断

词说一半 + 静音 + 重说完整：

| 原文             | 删除       |
| ---------------- | ---------- |
| 依赖[静]依赖关系 | "依赖[静]" |

### 4. 主动中断

明确表明是错了

| 原文                                     | 删除                                     |
| ---------------------------------------- | ---------------------------------------- |
| 这都，它没办法理解你到底是说错了，重新说 | 这都，它没办法理解你到底是说错了，重新说 |
