## 变更记录

Step2（语气词/卡顿词确认与规则更新）对规则文件的修改会追加记录到 `变更记录.md`。

## AI 审核顺序（按优先级）

1. **静音 >0.2s** → 建议删（按1秒格子拆分）
2. **残句** → 删（话说一半 + 静音）
3. **重复句** → 删短的（开头≥5字相同）
4. **句内重复** → 删 A+中间（A+中间+A 模式）
5. **卡顿词** → 删前面（那个那个、就是就是）
6. **重说纠正** → 删前面（部分重复、否定纠正、词被打断）
7. **顺句词/口误** → 这都、其实、对吧 等按 `10-顺句词删除.md`、`2-语气词` 移除
8. **语气词** → 标记待人工确认（嗯啊呃）

## 核心原则

**删前保后**：后说的通常更完整，删除前面保留后面。
