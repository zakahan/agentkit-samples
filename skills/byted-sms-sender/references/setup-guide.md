# Byted SMS Sender — 服务开通与凭证申请指南

本文档帮助从零开始申请和配置 Byted SMS Sender 依赖的火山引擎短信服务 API。

## 概览

```
注册账号 → 开通服务 → 创建消息组 → 申请签名 → 申请模板 → 获取凭证 → 配置环境变量 → 调用 API
  (1次)      (1次)      (1次)       (1次)      (1次)       (1次)        (1次)        (随时)
```

整个流程约 10~15 分钟。

---

## 第一步：注册火山引擎账号

> 已有账号可跳过。

1. 访问 https://www.volcengine.com
2. 点击右上角【注册】
3. 使用手机号注册(也支持飞书、抖音等第三方账号)
4. 首次登录控制台需完成**实名认证**

---

## 第二步：开通短信服务

### 2.1 进入控制台

访问：https://console.volcengine.com/sms

### 2.2 开通服务

1. 首次访问会提示开通服务
2. 点击【立即开通】
3. 选择计费方式(按量付费或资源包)
4. 完成开通

### 2.3 计费参考

| 服务 | 计费方式 | 价格 |
|------|---------|------|
| 国内短信 | 按条计费 | 约 0.04~0.055 元/条 |
| 国际/港澳台短信 | 按条计费 | 根据国家和地区不同 |


---

## 第三步：创建消息组

消息组是短信发送的基本单位,需要先创建消息组才能发送短信。

1. 进入 [短信控制台](https://console.volcengine.com/sms)
2. 左侧菜单选择【消息组管理】
3. 点击【创建消息组】
4. 填写消息组名称和描述
5. 记录生成的**消息组ID**(格式如: A123****)

---

## 第四步：申请短信签名

签名是短信开头的内容,如【火山引擎】。

### 4.1 申请签名

1. 左侧菜单选择【国内短信】→【签名管理】
2. 点击【申请签名】
3. 填写签名信息:
   - 签名内容: 如"您的公司名"
   - 签名来源: 选择对应的证明材料
   - 上传证明材料
4. 提交审核(通常1-2个工作日)

### 4.2 签名要求

- 签名内容需与证明材料一致
- 签名长度为2-12个字符
- 不能包含特殊符号
- 必须是已注册的企业、产品或网站名称

---

## 第五步：申请短信模板

模板是短信的主体内容,可以包含变量。

### 5.1 申请模板

1. 左侧菜单选择【国内短信】→【模板管理】
2. 点击【申请模板】
3. 填写模板信息:
   - 模板名称: 如"验证码模板"
   - 模板类型: 验证码、通知、营销等
   - 模板内容: 如"您的验证码是${code},有效期5分钟"
   - 关联签名: 选择已审核通过的签名
4. 提交审核(通常1-2个工作日)

### 5.2 模板要求

- 模板内容需明确、规范
- 变量格式: ${变量名}
- 不能包含违法违规内容
- 营销短信需包含"拒收请回复R"

### 5.3 记录模板ID

审核通过后,记录**模板ID**(格式如: ST_6f8c****)

---

## 第六步：获取凭证

### 6.1 获取 AK/SK

1. 登录 [火山引擎控制台](https://console.volcengine.com/)
2. 点击右上角**头像** → 选择「**API 访问密钥**」
3. 点击【**创建密钥**】
4. 系统生成一对密钥:
   - **Access Key ID**(AK): 公开标识符
   - **Secret Access Key**(SK): 私密密钥
5. **立即复制并保存 SK**(仅创建时显示一次,之后无法再查看)

> 安全建议: 不要使用主账号密钥,建议创建 IAM 子用户并授权。

### 6.2 子账号权限配置

如果使用 IAM 子用户的 AK/SK,需要主账号授权:

1. 主账号登录控制台
2. 点击头像 → 进入「访问控制」模块
3. 在「用户」页面找到子账号,点击【管理】
4. 切换到【权限】标签页
5. 点击【添加权限】
6. 搜索 `SMSFullAccess`,选中后确认

---

## 第七步：配置环境变量

```bash
export VOLCENGINE_ACCESS_KEY="your_access_key_id"
export VOLCENGINE_SECRET_KEY="your_secret_access_key"
```

建议将环境变量写入 `~/.bashrc` 或 `~/.zshrc` 以持久化。

---

## 第八步：验证调用

```bash
cd skills/byted-sms-sender
python3 scripts/volc_sms.py send_sms \
  --access-key "$VOLCENGINE_ACCESS_KEY" \
  --secret-key "$VOLCENGINE_SECRET_KEY" \
  --sub-account "A123****" \
  --signature "您的签名" \
  --template-id "ST_6f8c****" \
  --mobiles "13800138000" \
  --template-param '{"code":"123456"}'
```

如果返回结果中包含 `MessageIds` 等字段，说明配置成功。

## 常见问题

### Q: Secret Access Key 忘了怎么办?
A: SK 仅创建时显示一次,无法找回。需删除旧密钥,重新创建。

### Q: API 调用返回权限错误?
A: 检查:
1. 服务是否已开通
2. 子账号是否已授权 `SMSFullAccess`
3. 凭证是否正确

### Q: 签名/模板审核需要多久?
A: 通常1-2个工作日,加急审核可在几小时内完成。

### Q: 签名/模板审核不通过怎么办?
A: 查看审核意见,根据提示修改后重新提交。常见原因:
- 证明材料不清晰或不完整
- 签名内容与证明材料不符
- 模板内容不规范或包含敏感词

### Q: 短信发送失败怎么办?
A: 检查:
1. 签名和模板是否已审核通过
2. 手机号格式是否正确
3. 模板参数是否完整
4. 账户余额是否充足

### Q: 收到 RE:0006 手机号格式错误?
A: 检查手机号格式:
- 国内短信: 11位手机号或 +86开头
- 国际短信: 必须包含国际区号,符合 E.164 标准
