私网连接（PrivateLink）通过终端节点和终端节点服务建立私密的网络连接，实现不同 VPC 对服务资源的私网访问，避免了公网访问潜在的安全风险，为您提供更加安全的组网方式。
本页面介绍 VikingDB 私网连接使用方式的相关操作。
当前仅支持终端节点所属地域为华北2（北京）、华东2（上海）、华南1（广州）的服务使用方通过本页面操作进行私网连接。

# 前提条件
确保服务使用方已创建 VPC 及其子网。详细操作请参见[创建私有网络](https://www.volcengine.com/docs/6401/69467)。
# 操作步骤
## 创建接口终端节点

1. 使用服务使用方的火山引擎账号登录[终端节点控制台](https://console.volcengine.com/pl/endpoint)。
2. 未完成跨服务角色授权的账号需执行以下步骤完成角色授权。
   1. 在自动跳转的“跨服务访问请求”页面，单击“立即授权”按钮。
      授权角色在访问控制服务的角色列表中显示，删除授权角色将导致跨服务访问不可用，请谨慎操作。
      删除授权角色后，如需继续使用私网连接服务，请重新授权。

   2. 在左侧导航树重新单击“终端节点”。
3. 在顶部导航栏，选择目标项目和地域。
4. 在“接口终端节点”页签，单击“创建终端节点”按钮，参考下表配置相关参数。其中，私网连接方式下**类型**选择**接口**；**添加方式**选择**火山引擎服务**；
   **向量库-终端节点服务**：（可通过搜索 vikingdb 选择，**详见**[地域和访问域名（Endpoint）](/docs/84313/1792715)）

当前终端节点域名仅支持 https，无法使用 http。

5. 单击“确认订单”按钮，进入订单确认页面，勾选“我已阅读并同意《产品和服务协议》《私网连接服务条款》《私网连接服务等级协议》”。
6. 单击“立即购买”按钮，跳转到申请提交成功页面。
7. 单击“去控制台”按钮，进入接口终端节点列表页面，查看已创建的接口终端节点实例。
8. 创建终端节点完成后，会自动创建出私有 DNS，因此您仍可使用公网域名访问，DNS 会自动解析为私网 IP 地址进行请求。
